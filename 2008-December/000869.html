Hi,
I have a setup like this: all my music is on a samba share, mounted 
as ~/music locally.
MPD runs locally.

Now, let's say I want to edit tags on a few songs that are in 
~/music/new/fresh_tracks

After I'm done editing, sonata instructs MPD to update the whole 
database. This can take up to an hour, because I have tons of music 
files.
When I edit the tags in a different tag editor and then update just the 
new files with 'mpc update /new/fresh_tracks', it takes a few seconds 
at most.
