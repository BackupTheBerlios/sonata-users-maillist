The reason why I have started this thread was to think about a structure
that could be used in the future for the plugin. I run MPD on a local
machine which has two active users. I controll MPD in two ways. One is using
Sonata, when I have my desktop on and the other one is via my mobile phone,
anyremote and mpc.

I needed basically two things. To be able to start the loop via mobile phone
and end it in Sonata and vice versa. To give the user a possibility to
cancel the loop when other user  accidently forgets it on. I think that MPD
with its client-server approach is a great tool to achive this, and I like
Sonata very much on the desktop for its clean interface and rich feature
set.

Many thanks a Scott and others for developping Sonata.
