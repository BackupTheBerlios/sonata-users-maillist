On Wed, Dec 24, 2008 at 8:02 AM, Dan McGee <dpmcgee at gmail.com> wrote:

> What exactly are you changing the behavior to? If I can't issue the
> equivalent of an mpd update though Sonata, that would be a regression
> to me...
>

The default has always been that if you issue an update via Ctrl-U or the
right-click menu, it asks mpd to do a full update on your entire library. So
even if you select a directory or set of files and ask for an update, sonata
would update the entire library, which can be slow for people. You had to
press Ctrl-Shift-U if you only wanted certain files/paths updated.

Now I've swapped behaviors. Ctrl-U or using the right-click menu will only
update selected files/paths. Ctrl-Shift-U will update the entire library.
