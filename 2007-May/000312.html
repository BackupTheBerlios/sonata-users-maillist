Hy List,
Sonata currently uses egg.trayicon to display the trayicon by default.
It seems this module hast been deprecated:
"~/devel/sonata/sonata.py:58: DeprecationWarning: the module
egg.trayicon is deprecated; equivalent functionality can now be found in
pygtk 2.10"

I think it would make sense to use the pygtk functionality by default
and only fall back to egg.trayicon if pygtk is < 2.10.
A patch that would do this is attached.
