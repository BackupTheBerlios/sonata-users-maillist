On 5/26/07, Florian Zeitz <florian.zeitz at gmx.de> wrote:
> It seems this module hast been deprecated:
> "~/devel/sonata/sonata.py:58: DeprecationWarning: the module
> egg.trayicon is deprecated; equivalent functionality can now be found in
> pygtk 2.10"

I had filed a bug about this against gnome-python[1], it seems they
weren't aware that gtk.statusicon actually _doesn't_ provide
equivalent functionality. So it appears that pygtk will be
undeprecating the module (although if more folks want to throw their
support behind the decision, please add your comments to the bug
report). There are also separate feature requests filed on the
gtk.statusicon to provide some of the features that only egg currently
provides.

Long story short, it looks like those people who use egg.trayicon with
sonata will most likely be able to continue using it ;-)

Scott
