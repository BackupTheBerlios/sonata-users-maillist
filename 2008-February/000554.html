> Yes. It could be useful to be able to drag a collection file from nautilus/konqueror/... and insert it where you want in the playlist.

Okay, this is implemented in svn. Note that you also need mpd-svn
because the only way this can be done is using mpd's improved 'addid'
command (as of a few commits ago). You'll be able to drag-n-drop both
files and directories; the latter will be recursive.
