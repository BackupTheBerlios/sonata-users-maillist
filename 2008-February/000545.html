If you have them all at 64px it you will see less items and the same
time and it will be harder to get a quick overlook (then again you
should know what music you have :-p). So for that reason I think it
would be nice if you at least had a few options to choose from
(current size, 32, 64 or something). I'm mainly thinking about how the
monitor on my laptop is pretty small and there I'd like to see more
items at once and hence want smaller icons. Though myself I don't
spend much time in the library, but other users might.

If it's easier to have the user just put in the value he/she wants
than a bunch of fixed values then perhaps go with that right away.
