Just a couple things to point out:
1. Not a big deal, but you can use os.path.splitext to retrieve a
file's extension.
2. I'd probably allow ftp in addition to http. I'd imagine it works fine.
3. You probably want to use a more flexible approach than hardcoding
in eligible image extensions. Something like this should do:

def image_is_valid(self, extension):
     for i in gtk.gdk.pixbuf_get_formats():
          if extension.lower() in i['extensions']:
               return True
     return False
