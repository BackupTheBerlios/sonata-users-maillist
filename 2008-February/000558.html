Hi.

The idea is pretty good indeed but I have a problem with the current
implementation... As the compositing of the case image is done on the
whole original cover image, the opaque part of the case image always
hides the left part of the original cover. While it may not be a
problem for some images, it gives quite an horrible result if there is
something to see on the left side (cf.
http://pix.nofrag.com/b/d/a/b15fb3e9f4a6fc1c890aa2257ba00.html for an
exemple).

Wouldn't it be possible to resize the sonata-case.png image so that it
mimics the shape of a real CD case (the case is a rectangle, not a
square and only the transparent part is a square) ? Obviously, this
means the code may become more complex so it may not be feasible...
