Hi John,

John wrote:
> I just not long ago bought an Acer Aspire One netbook running Ubuntu 
> 8.10 eeepc, and would like to install this on there.

Ubuntu 8.10 (Intrepid) includes Sonata version 1.5.2 in its Universe 
repository so you can point-and-click install it using the Synaptic 
Package Manager. If there's enough interest within the Ubuntu community, 
they can include newer versions in their Backports repositories (they 
did this for Sonata 0.12.1 back in Ubuntu 6.06).

If you're willing to venture outside Ubuntu support and try out Sonata 
1.6, you can download and extract 
http://download.berlios.de/sonata/sonata-1.6.tar.gz and run the program 
sonata.py within.
