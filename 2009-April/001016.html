Thanks for the clear problem description :-) As you're new to Ubuntu, 
you should find the documentation here useful in general: 
https://help.ubuntu.com/

Regarding this specific issue: A normal user account in Linux doesn't 
have the permissions to modify system files, hence the x. Further, the 
system cannot always guess and you have to choose *which* application 
you want to open a file in. You do this by right-clicking the file icon 
and choosing "Open with other application..." or such.

You can click "Text editor" in the list once, and see its name "gedit" 
appear in the "Use a custom command" section. To edit with root 
(administrator) capabilities, prepend "gksudo" so that you get
"gksudo gedit". Now you can click "Open" and the file should open in a 
Text Editor after you provide your password.

Now that you know this, the more powerful, alternative way is to open a 
Terminal window and enter "gksudo gedit /etc/mpd.conf".
