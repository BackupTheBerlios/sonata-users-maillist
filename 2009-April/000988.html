I'm not a fan of the Firefox sidebar. In order for it to be really useful --
to keep you from having to constantly scroll left/right to read things and see
what's going on -- you have to keep it set wide. And if it's set wide, then it
covers a great deal of the main window -- which now means you have to resize that.

torn off the main window, into its own separate window. Tuukkah mentions

The xplorer2 file manager uses an interesting model that might give us some
ideas. It's a two pane file manager, and you can open directories in tabs in
both panes. The nice thing about the functionality is that unlike the new
Nautilus implementation of tabbed file management, you can DnD files onto a
tab. (Nautilus simply switches to the tab and forces you to drop your files in
the directory window.)

DnD just switching to the Current tab and then requiring you to scroll the
list to find your drop spot may be fine with short lists, but if you have a
huge list (some of the screenshots on the Sonata website show thousands of
songs in the list), it's just impractical.

Directories, obviously, are automatically sorted, so there's no need to drop
things in a specific spot in a list, as you would with a playlist. (Except in
the Nautilus implementation where you might have to scroll past a list of
directories.) It seems that there might be an easy way of handing this issue
in the Sonata UI.

If it was possible to set a "marker" in the Current list, then the DnD on the
Current tab could drop the files at that location. (ALT-click might be a good
idea, since it's not used; and it should probably set a different color
highlight than normal left-clicks.) And if there's no marker, the add is just
to the bottom of the list.

This marker system would allow you to quickly scroll through your Current list
to see where you want the new files to be dropped, and then mark it. It could
also be used in conjunction with the current double-click or right-click+Add
functionality.

There could be an option to automatically switch to the Current tab after a
DnD operation, since some people might want to then do some further
rearranging of their list.

Anyone think that any of that might be suitable to Sonata?
