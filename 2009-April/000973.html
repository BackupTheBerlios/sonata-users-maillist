In general, it's easier to reply to you if you provide a more concrete 
description of your situation: the steps you took, the results you saw, 
the results you were hoping for.

> I have tried to 
> install the Ubuntu Synaptic manager, but I cant get it to connect. I 

1. Do you mean you don't have Synaptic on your computer? In this case, 
either you're not running normal Ubuntu or it's broken.

2. Or, if you have Synaptic, do you mean it doesn't work at all? In this 
case, your Ubuntu system is broken and needs fixing.

3. Or, if you were able to install Sonata, do you mean you can start it 
but you can't get it to connect to MPD? In this case, you're halfway 
there. You should check that you have MPD installed and configured and 
that it's running.

> also then tried to install by downloading the other version, but when I 
> clicked on the sonata.py file, nothing happened.

If Sonata doesn't install and work correctly from the Ubuntu packages 
first, it most probably doesn't help to try versions outside Ubuntu 
either. The reason to try would be if you had an older version of Sonata 
and wanted to try the newest one.

You can check whether you get something about Sonata in the 
.xsession-errors file in your home directory, and you can try to run 
sonata.py from a Terminal window (enter the full path, 
/home/john/sonata-1.6/sonata.py or so) and see if you get any error 
messages.


Hoping this helps,
Tuukka
