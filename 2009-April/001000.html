John wrote:
> Ok, well, I installed mpd, but I still cant work out how to set it up to 
> play the music. I did a bit of a search on the Ubuntu forums, and on 
> there there are quite a few people having the same problem. I couldnt 
> find where you set up mpd. Can you help please?

Unfortunately, you don't actually give concrete information about the 
steps you took and the problem you are seeing, so I have to guess...

1. You edit the file /etc/mpd.conf. At least, you should change the 
value of music_directory to point to your music collection. For more on 
mpd.conf, see http://mpd.wikia.com/wiki/Configuration

2. You restart mpd with the command "sudo service mpd restart"

3. *If* you can connect to MPD from Sonata, see your music collection, 
add songs to the Current playlist, and start playing, but it stops right 
away with no sound, *then* you have hit the problems with Ubuntu's 
PulseAudio. For help, see 
http://mpd.wikia.com/wiki/PulseAudio#Version_0.12.0_and_later
