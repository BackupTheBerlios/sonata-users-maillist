I enabled the plugin system again in SVN, and did some work on it, 
although it remains to be seen whether it could be kept enabled for the 
next release.

Each plugin is now disabled by default and they are not loaded before 
use. The plugins can be enabled in the tab in the preferences window.

I've included a test plugin that shows the capabilities and how to 
program to them, plus a Local MPD plugin, "A tab for controlling local 
MPD" (currently it just displays some info, but I hope to add a button 
for "gksudo gedit /etc/mpd.conf" and "gksudo service mpd restart" ;-).

The currently implemented capabilities:
* "tabs": Before today, this was the only one: it lets the plugin add 
its tabs.
* "playing_song_observers": This lets the plugin react to song change 
like the album art, lyrics etc. do.
* "enablables": This lets the plugin do some setup and teardown work.

If you're writing a plugin and need something more, let me know and 
we'll see if it's reasonable to implement at this point or if it needs 
to wait until later.
