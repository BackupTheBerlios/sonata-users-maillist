Thanks, a cool example of what's possible! Did you hit any issues while 
developing this, or was it all easy?

Some comments:

Would it make sense to remove the construct_tab capability, as we don't 
want too many tabs? Do we want something more than the "Enabled" 
checkmark - to signal that the plugin is working fine?

I think there's a slight bug in on_enable. Currently, if you disable and 
enable the plugin, state will be True but tune will not be None, and the 
tune info will be cleared. Would it make sense to destroy the D-Bus tune 
object and set tune to None when the plugin is disabled?

Other than that, I think it's a nice little plugin, like they should be. 
Do you mind if we include this in the Sonata SVN and releases? I suppose 
each plugin should contain "license: GPL v3 or later" in the [plugin] 
section.


Thanks again,
