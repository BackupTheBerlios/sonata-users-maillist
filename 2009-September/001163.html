This is what I did -- having never used git before -- and it worked fine...

1) created a sonata directory
2) from a term inside that dir: git clone git://git.berlios.de/sonata
3) that created another sonata directory and cloned the source tree
4) did git pull from the same dir I did the clone from and got the error 
you did
5) did git pull from the directory the git clone created and it worked fine.

Hope that helps...
