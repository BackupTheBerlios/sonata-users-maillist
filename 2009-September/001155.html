With great sadness I must announce Scott's departure from the project.
For three years Scott has crafted Sonata into what it is today, a simple
and elegant music player. I hope we can carry on this tradition in the
future.

Over the past few months, activity has dropped off and Scott has decided
to hand over the project to Tuukka and I. I've decided to step-up as the
new maintainer as I'm still using Sonata and actively working on the code.

I'm taking this opportunity to make some changes. I'm planning on
switching our version control system from svn to git. I'm currently in
the process of converting the svn repository to git and cleaning it up a
bit. I'm going to use this conversion process to institute a spaces only
policy in python code and to split off the website into its own branch.
I'm also going to go through the administrative things and cleanup our
berlios.de site (close bugs, patches, etc.)

We'll still be using berlios.de for our hosting so finding us shouldn't
be hard. Thanks for staying with us through this transition period.
