Hello,

I just tried the latest svn, real-time library search is almost
instantaneous. Even when using the 'everything' search field, it's as
fast as playlist search here. Bear in mind, though, that I run mpd on
localhost like you. Here are my mpd stats:

3897 songs
348 albums
418 artists

The only issue I noticed is this:

If I want to search for the song "Everybody's gotta learn sometime" by
Beck, for example: In the playlist search I would start typing the
Artist name followed by the song name, i.e. "beck everybody" and the
song would show up. However if I do the same in the library search I
type the Artist name and then it doesn't find anything after that,
typing the song name, or even just typing a space. I use the filesystem
view in the library, though the same behaviour is there with both the
artist and genre views.

It is obviously a minor issue, as I can just type the song name instead
of the artist's name followed by song name, but I think it's better to
have consistent behaviour accross playlist search and library search.

I have to say, I like real-time library search even more now that I
tried it.
