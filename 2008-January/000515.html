Hi

I have created a patch (my first ever one!) to allow sonata to use the
transparency with the new murrine engine. Please see the screen shot at

http://idnzor.net/tmp/Sonata.png

As you can see the tab buttons on the main window seem to be a bit
funny, this is not the engine but I think how the tabs are created. 

Please find the patch at

http://idnzor.net/tmp/rgba.patch

I am not sure if this thread can accept attachments - never emailed a
mailing list before :)

For more info on the murrine engine and transparency see:

http://www.cimitan.com/blog/2007/12/12/gtk-rgba-transparent-widgets-with-the-murrine-engine

This is all 100% degradable - so users without a composite window
manager will see no difference, it is also super fast still! You will
need the latest version of murrine which can be found at 

http://bazaar.launchpad.net/~cimi/murrine/trunk

Example: bzr branch http://bazaar.launchpad.net/~cimi/murrine/trunk

Look forward to your feedback

Jon Bolt
