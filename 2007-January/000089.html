I've built an Ubuntu Edgy package for the 1.0 release of Sonata.  i386
and amd64 builds as well as the sources are available here:
  http://packages.matt-good.net/

This version is based on the official Debian package of Sonata 0.9.  The
Sonata download page should probably link to the Debian page since
Sven's packages are missing some dependencies:
  http://packages.debian.org/unstable/sound/sonata

The i386 package on my site was built via pbuilder in an i386 chroot on
my amd64 machine.  This is the first time I've tried this, so if for
some reason it doesn't work let me know.
