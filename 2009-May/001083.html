Hello everybody,
I would like to share one idea, that I am using, maybe someone would be
interested to write a plugin like this. Unfortunately I cannot do it,
because I know nothing about python :-(..

The idea is to be able to set time bookmarks in songs. I am using this when
I want to learn a song, to come back to a particular spot. But I can be used
for word transcription or translations as well. At the moment I am using a
set of bash scripts. One sets the bookmark, the other jumps to this
bookmark. I am using fluxbox, which has excellent support for keybinings, so
I am limited only by my on memory when setting the keys to set and call
bookmarks, but other managers do not have this possibility (like metacity),
and here is where the Sonata's plugin could be useful.

I played with the scripts a little, so now I am able to set the bookmarks
per song (songs must be properly named, though), and the bookmarks are
remembered even after restart.


Here is the script that sets the bookmark:
=================================================================
tomas at Ubuntu-ElPraga:~$ cat /usr/local/bin/MPD-Q-mark.sh
#! /bin/sh

# A simple script for setting time bookmarks for every song
#
# Tomas Kaluza

MARK=`mpc |grep / |grep % |cut -f5 -d" " | cut -f1 -d"/"`
SONG_NAME=`mpc |grep -`
MARK_FILE=MPD-Q-mark

mkdir -p "$HOME/.MPD-bookmarks/$SONG_NAME"
touch "$HOME/.MPD-bookmarks/$SONG_NAME/$MARK_FILE"
echo $MARK >"$HOME/.MPD-bookmarks/$SONG_NAME/$MARK_FILE"

echo Setting Q mark for $SONG_NAME at $MARK
#zenity --text "Setting Q mark for $SONG_NAME at $MARK" --info --timeout 5
2>/dev/null

=================================================================


Here is the script that calls the bookmark:
=================================================================
tomas at Ubuntu-ElPraga:~$ cat /usr/local/bin/MPD-GOTO-Q-mark.sh
#! /bin/sh

# A simple script reading a time bookmark for a song
#
# Tomas Kaluza
MARK_FILE=MPD-Q-mark
SONG_NAME=`mpc |grep -`



mpc seek `cat "$HOME/.MPD-bookmarks/$SONG_NAME/$MARK_FILE"`

=================================================================

I am using Win-Q, Win-W, Win-E, Win-T to set the bookmarks (each calling own
script) and Win-A, Win-S, Win-D, and Win-F to call the bookmark.

I guess that from within Sonata it could be done in even cleaner and nicer
way, but this works for me fine :-)

I hope someone finds this useful..

Regards

Tomas
