Hello,

I'm reposting this request, maybe it can be implemented as a plugin.
My idea is that there be a 'selected songs' options under the 'Current' tab.
It would have 'save to playlist' and 'add to playlist sub-options. Save to
playlist will create a new playlist with the selected songs, add to playlist
will add the selected songs to an existing playlist.

I need this because I have a 'misc' playlist containing various songs, and I
as go through my library, I add new songs to 'misc'. Currently, I do this: I
delete all songs from Current, except the one that I want to add. I go to
playlists and I add the 'misc' playlist. Then I go to Current again and I
save the playlist as 'misc'.
A more elegant solution for this would be apreciated.
