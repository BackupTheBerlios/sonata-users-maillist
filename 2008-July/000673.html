I have noticed some changes of behavior in Sonata when using the library search.
I am using sonata-1.5.2 (although it reports to be 1.5.1 in `sonata --version`)
on my laptop and an (old) sonata-1.2.1 on my desktop.

I connect to the same mpd server with both versions and the output varies:
- 1.2.1: the songs are displayed sorted (by artist-album-tracknum)
    (or maybe by filename because I use renaming tools for my music dir)
- 1.5.2: no sorting is applied

I think that it is totally inconvenient in the scenario of searching
for an artist name and
wanting to find "the n-th" song of the "Somename" album.

I know there is alternative way of doing this with Sonata, but thought
it should be considered
by the dev.
