Hey there, I made these icons a bit back.  They use the Tango! project's 
colors and general style (I can tweak if necessary) and (imo) make Sonata look 
a little snazzier, especially in the system tray.  As they're directly based 
on the current Sonata icons, I'm releasing them under the GPLv3, so you guys 
feel free to use them however you see fit.

A couple notes, if you're interested in using them with sonata:
The 22x22 icon does look a lot better in the systray and the larger icons look 
better as the actual app icon, but Sonata currently just looks at one size of 
icon in /usr/share/pixmaps.  One way to kill two birds with one stone would be 
to adopt a more standard system for the icons in the hicolor directory.  That 
way, not only would you be able to specify different resolution icons in the 
future more easily, but individual icon themes could use different sonata 
icons.

I'm not too familiar with mailing lists, so hopefully I can just attach this 
gzip'd tarball.
