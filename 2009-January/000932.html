I asked Scott and he preferred to start by default, so I added an option 
--no-start for the opposite.

Further, --toggle now shows the main window even if Sonata needs to be 
started and it was hidden last time it was running. Analogously, --popup 
now hides the main window even if Sonata needs to be started and it was 
visible last time it was running.
