>   Why not install mpc (a lightweight command line client) and use

> each different in their way.  You can have a nice GTK+2 app, and on

Configuring multimedia keys in GNOME works for each event named above;
applications, e.g. Rhythmbox, Banshee, Totem, each take turns in
responding to them as appropriate.  (The active window gets first
crack, then whichever media application has control, as determined by
some indeterminate magic.)

Adding support in Sonata to grab these shortcuts would be
significantly easier than trying to configure mpc to respond to these
keys, as one cannot have both (AFAIK).  Plus it'd just be easier for
the user. :)
