1)
What would you guys say if the tray icon would accept more mouse-clicks than just left and right? Say.. wheel = next/prev song ; middle-click = play/pause etc. User defineable options would probably be essential for a thing like that?

2)
I don't know how many of us keep changing the status of repeat/random more or less often, but I'd find it more user friendly to have those and a crossfade toggler too (still waiting for that one .. ;) ) in the tray's right click menu. I also find the <show sonata> menu entry kind of needless; if one can do a "right click icon -> left click <show sonata>" I think it's not a problem to only "left click icon" to achieve the same result. I don't mind it being there, but as I said, I just find it quite needless.

3)
Also I would find a user defineable popup "tooltip" for the tray icon really nice. I like to have the cover art and status bar shown in the main window, but would prefer a simpler popup for the tray.

4) Kind of related to 3)
A way to display the song time in the main window/tooltip without the status bar would be nice, though I know it leaves one without the possibility to skip to song position, but disabling the status bar does it anyhow..

So a few thoughts there, anything that rings a bell?
