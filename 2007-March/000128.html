On 3/29/07, Scott Horowitz <stonecrest at gmail.com> wrote:
> I was thinking about this some more on the bus ride home from work,
> wondering if there might be some sort of algorithm that would make 99%
> of people happy. It seems to me that a Various Artists album would
> tend to never have more than some maximum number of songs from the
> same artist - say 5.
>
> So if we had Song A from Artist A on the Untitled Album.. we would do
> a search for all other songs with Untitled Album as the album name. If
> these songs span only a single artist, Artist A, it's a simple case.
> If instead there are multiple artists, we check to see if there are
> more than 5 songs by any artist. If so, it's a good chance that we are
> dealing with multiple albums from different artists (i.e. Untitled
> Album by Artist A and Untitled Album by Artist B). If not, we assume
> this is a single Various Artists album.
>
> Anyway, just wanted to jot down these thoughts.. if anyone else has a
> suggestion, feel free to mention it ;-) I'm sure there are other ways
> you could structure the algorithm. But I think something along these
> lines would be an interesting approach..

I was thinking of something reallly simple, when selecting album art,
you could add a checkbox (or something) that states "this is a mixed
artist album" and the file would be saved as album.jpg. When looking
for the album art you first check if there is any artist-album.jpg
file, if not, check if there is an album.jpg file if so use that.
Though the downside is you have to set it up manually. :-/

While looking around I found out how BMPx is doing it:
http://forum.beep-media-player.org/viewthread.php?tid=438

I'm not sure I like their approach since it again will also lead to
manual work by setting the extra tag up.

Amarok seems to user the same approach as Soanta (but using a hash of
artist-album as name):
http://amarok.kde.org/forum/index.php?topic=11741.msg11871#msg11871

And they seem to have an internal database that keeps track of VA
albums: http://amarok.kde.org/forum/index.php?topic=10146.0

That's what I've found out.. I've never used many players myself (only
XMMS/Audacious) before moving to MPD and Sonata so I don't know many
others.
