> this tag in the GUI of the

Newer versions of MPD (>=0.15) have a "sticker database" which could be
used for song ratings. I think a song rating feature would be a nice
addition.

With respect to the comment tag, I don't see a reason why not to include
this in the formatting. I'm working on a patch to dynamically gather the
supported tags from MPD by using the "tagtype" command. This command is
supported in versions of MPD >=0.13 so hopefully we can bump the
requirements to at least 0.13 next release.
