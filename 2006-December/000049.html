Hello!

I've been using Sonata for about a week now, and I must say, you've done
a terrific job. It's easily the best MPD client I've tried. It's very
pretty, and the interface is both intuitive and clean. 

I have a feature suggestion. You might not find it fitting for Sonata,
but I just thought I'd mention it. 

I find it a bit time consuming to browse my music library, since it is
pretty nested and large, and I propose a few alternative browsing
methods. My idea is that you should be able to browse by Artist, Album,
File system hierarchy (like we currently do) and possibly Year and
Genre. I have made a mockup to illustrate my point:
http://tinyurl.com/y42ha2

Perhaps the search function could be merged into this as well?
