I've written a patch to add LyricWiki support to sonata.
Right now getting lyrics is very slow. LyricWiki seems to have some
problems with a rapidly growing number of users. I hope they solve this
soon.
It gets the lyrics of all songs that are currently selected.

Disclaimer:
The patch makes heavy use of threading, so it shouldn't block the UI at
any point, BUT this is the first time i used threading and PyGTK, so
this patch might be buggy and some people might consider it hackish.
