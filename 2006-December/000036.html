I've recently swicthed over to MPD and Sonata (great client btw!), and I've
found that I really miss the search function availabe in XMMS. Browsing the
MPD wiki I found the following hack
http://mpd.wikia.com/wiki/Hack:mpdfindwhich is a small python script
implementing a search just like the one in
XMMS. It would be great if Sonata could have a similiar search function, the
current one is pretty useless to me since I have to type in both the artist
name, hyphen and then song title.. and if I'm lucky the search box stays
open long enough for me to enter all the info. But.. when I have several
songs with the same name (ie studio recording and live recordings) it'll
stick with the first match found, not so great if I want to hear one of the
other matches.

I'd also like to be able to edit the tags of the file though I think that
would require some work since the different music formats use different tag
formats. Showing the length of the songs in the playlist would be nice
(aligned right if possible would be even nicer).
