On 12/6/06, [vEX] <niechift.vex at gmail.com> wrote:
> Would it be possible to add an option to show the track number as well
> as from which album the current song is from? And only show it if
> album information is available.
>
> Something like this:
> <song>
> by <artist> from <album>, track <#>

In svn, you can now enclose formatting in parentheses to dictate that
you only want that information to appear if all tags within the
parentheses are found. So, for example "(%T_)%A - %S" would show
"04_Ben Harper - Alone" if the track number exists and "Ben Harper -
Alone" if the track number doesn't exist.

I don't yet have the current song label available for formatting
though (it's on the todo), but this can be used in any of the existing
formatting strings thus far.
