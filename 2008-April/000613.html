Dear Sonata-devs,

I use Sonata both as a GUI and I've bound the CLI options to the multimedia
keys on my keyboard.

Using "sonata pp" caused a bit of a lag, so I did a simple comparison of
mpc versus sonata:

$ time mpc toggle
Faith No More - The last to know
[paused]  #13/14   2:43/4:27 (61%)
volume: 42%   repeat: on    random: off

real    0m0.084s
user    0m0.024s
sys     0m0.004s

$ time sonata pp

real    0m1.539s
user    0m0.304s
sys     0m0.080s

(AMD Athlon64 X2 Dual Core 3800+)


Maybe it would be a good idea to include a simple(?), CLI-only tool with
Sonata for fast execution?
