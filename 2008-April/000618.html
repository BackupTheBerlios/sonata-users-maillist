On Thu, Apr 17, 2008 at 3:39 AM, Tomáš Kaluža <elpraga at gmail.com> wrote:
> I have one suggestion. In current versions of Sonata there is not an easy
> way to remove one or more song from a current playlist. The current menu
> ¨Write to playlist¨ is adding the songs into the playlist. Which can be
> useful sometimes, but when I try to erase a song from a playlist I use, I
> load it, erase the song, and write it again, I end up the all the song being
> doubled, and the one I wanted to erase remains single in the playlist.

I've gotten a fair bit of feedback about this behavior, so I'm willing
to improve it.

I'm definitely not a fan of increasing the number of menu items. That
said, I think a nice compromise would be, upon clicking "save to" >
"my playlist", to prompt the user about whether they want to overwrite
or append to the playlist. We can also save the user's choice to make
it the default the next time they get the popup so the user only has
to press enter.

Scott
