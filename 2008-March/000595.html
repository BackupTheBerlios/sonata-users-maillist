Hi,

the problem is solved. When I first encountered the problem with äüö I 
tried several things and always recreated the mpd database by mpd 
--create-db and never had success.

What I realized only now by try to use a single folder on a different 
hdd was, that this command didn't do anything, although in urxvt I saw a 
lot of files that seemed to be added.

What tried then was /etc/rc.d/mpd create-db and only by this command the 
database really was updated.

So, I recreated the database with the options of  which I thought they 
were correct and now I have no more problems with öäü.

Thanks for the hint that opened my eyes :)

Thomas
